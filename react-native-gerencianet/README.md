# react-native-gerencianet

A module to integrate gerencianet api sdk for android

## Installation

```sh
npm install react-native-gerencianet
```

## Usage

```js
import { multiply } from "react-native-gerencianet";

// ...

const result = await multiply(3, 7);
```
